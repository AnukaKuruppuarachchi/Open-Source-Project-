## Project Governance

Any decisions regarding the development, direction, and overall management of the project are made by the maintainers.

The maintainers of this project are responsible for:
- Reviewing and merging pull requests.
- Overseeing project direction.
- Ensuring the code of conduct is maintained.
- Responding to security reports.

## How contributors can become maintainers. 
If we see a contributor who’s active in the community and consistently making high-quality contributions, we’ll contact them with an invitation. 
High-quality contributions usually include: 
- Positive community engagement
- A strong understanding of the project
- Consistent participation
- Helpful suggestions that improve the project overall

Want to become a maintainer but haven’t heard from us? Email  OSP@email.com.
