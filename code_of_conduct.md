# Code of Conduct

## Purpose

This project aims to provide a collaborative and respectful environment for everyone involved in its development. All contributors are expected to behave in a professional and courteous manner when participating in the project.

## Expected Behaviour

All participants should:

- Be respectful and considerate in communication.
- Treat other contributors with professionalism and courtesy.
- Provide constructive feedback when reviewing code or discussing ideas.
- Be open to suggestions and collaboration.
- Focus on improving the project through positive contributions.

## Unacceptable Behaviour

The following behaviour will not be tolerated:

- Harassment or discrimination of any kind.
- Personal attacks, insults, or offensive language.
- Dismissive or hostile behaviour towards other contributors.
- Any conduct that creates an uncomfortable or unwelcoming environment.

## Communication Guidelines

When interacting with others in:

- GitHub Issues
- Pull Requests
- Discussions
- Meetings or messaging platforms

contributors should maintain respectful and constructive communication at all times.

## Reporting Issues

If a contributor experiences or witnesses behaviour that violates this code of conduct, they should report it to the project maintainers.

Project maintainers will review reports and take appropriate action if necessary.

## Enforcement

Project maintainers are responsible for ensuring that this code of conduct is followed. If unacceptable behaviour occurs, maintainers may take action including:

- Requesting changes to communication or behaviour
- Removing inappropriate comments or contributions
- Restricting participation in the project if necessary

## Acknowledgement

By participating in this project, contributors agree to follow this Code of Conduct and help maintain a respectful and collaborative environment.
